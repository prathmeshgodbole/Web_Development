Functioning

  • How does Trackji work ?

    Trackji collects all the data from the targeted device and sends them to your admin panel. For this Trackji application must be installed on the targeted device. The targeted device must have Internet access and must be compatible with the application. Internet connection is required to install a new device, then if the connection is lost, all data is stored and transmitted when the connection is back.

  • Can you help me track a phone based on its number ?

    It is not possible to help you to track the phone using only number.

  • What is the frequency of updating data ?

    If the device have the active internet connection Trackji transfers the data immediately. In absence of the internet connection, the data will be saved on the targetes device and the data is transferred as soon as the device gets the active internet connection.

  • Where can I find the transferred data ?

    To access the transferred the data you have to log in your account/admin panel provided by Trackji.

  • Is it necessary to have physical access to the target device to install Trackji?

    Yes, it is nessary to have physical access to the target devices till the installation of application, after installation Trackji will send the data automatically to your admin panel.
